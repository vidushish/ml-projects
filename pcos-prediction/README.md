# PCOS Prediction Model

This project implements a machine learning model to predict Polycystic Ovary Syndrome (PCOS) using clinical and hormonal features.

## Problem Statement

PCOS is a common hormonal disorder. Early detection can help in timely treatment. This model aims to assist in prediction using patient data.

## Features Used

* Age
* BMI
* Menstrual Irregularity
* Testosterone Level
* Antral Follicle Count

## Approach

* Data loading using NumPy
* Checked for missing values
* Split data into training and testing sets
* Trained Logistic Regression model

## Results

* Accuracy: ~88.5%
* Evaluated using confusion matrix

## Tech Stack

* Python
* NumPy
* scikit-learn

# Student Performance Prediction Model

This project implements a machine learning model to predict student performance based on academic and behavioral features.

## Problem Statement

The goal is to predict a student's total score using factors like study hours, attendance, and class participation.

## Features Used

* Study Hours
* Attendance (%)
* Class Participation

## Target Variable

* Total Score

## Approach

* Loaded dataset using NumPy
* Removed unnecessary columns and handled missing values
* Selected relevant features and target variable
* Split data into training and testing sets
* Applied Linear Regression model
* Improved performance using feature scaling

## Results

* R² Score: ~0.66
* Mean Squared Error: ~80

## Tech Stack

* Python
* NumPy
* scikit-learn
